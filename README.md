# Friends of TUT - Student Creative Movement

## Project

Friends of TUT website - Fashion, Media, Music at Tshwane University of Technology.

## Technologies

- Vite + React + TypeScript
- shadcn/ui
- Tailwind CSS
- Backend: Express + Nodemailer (Ethereal demo)

## Run locally

```sh
npm install
npm run dev
```

Backend:
```sh
cd backend
npm install
npm run backend
```

## Features
- Instagram, phone, email contacts
- Application form with backend email
- Responsive Wix-inspired design
